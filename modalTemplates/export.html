Import / Export

Download Json to computer
Export all the DataGroups to a JSON file on this computer. This can subsequently be imported into the app using the import tab.
Download TSV to computer
Export all the DataGroups to a TSV file on this computer. Used for importing into a spreadshseet. Not yet enabled.

Importing into an existing Collection. DG's from the import will be merged into this one. If there is already a DG with the name as in the import it will be replaced.
Creating a new Collection which will have the same contents as the imported one, including the name.

It won't be saved in the repository automatically - you'll need to click the save button. If there is already a collection in the repository with the same name then there will be 2 collections with the same name, but likely different contents.

A collection name can be changed on the 'Collection Info' tab on the main editor page.

The file selected MUST be an export file from the export tab in this dialog
