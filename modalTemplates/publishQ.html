An interim release can be done at any time - usually to allow others to review it. The minor version is incremented. A formal publication is when a Q is finalized (at least a specific version) and will increment the major version.


Status
The status is generally set with the release type. For an interim release it will be 'draft' and for a publication it will be 'active'.
